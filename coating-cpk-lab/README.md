# Coating CPK Lab

一个面向锂电涂布/检测数据的轻量级浏览器工具：可以模拟面密度或膜长数据，注入典型制造异常，自动计算 Cp/Cpk，并生成客户汇报型 FACA 草稿。

这个项目的定位不是“又一个图表 Demo”，而是把制造现场最常见的质量问题做成可交互的实验台：

- 调整过程波动、中心偏移、趋势漂移、泵磨损、气泡/漏箔冲击等参数；
- 立即观察趋势图、直方图、规格线、均值线和异常点；
- 自动计算 Mean、Sigma、Cp、Cpk、OOS 数量、OOS PPM、Rolling CPK；
- 根据数据特征给出异常类型判断；
- 一键生成 FACA / 5Why / 围堵措施 / 验证计划草稿；
- 支持导入 CSV、导出 CSV，便于把 MES、AOI、β-ray 或 CCD 数据接进来。

## 为什么有用

锂电制造过程里，很多质量讨论并不缺“数据”，缺的是把数据快速变成判断：

- 当前 CPK 变差是均值偏移，还是 σ 扩大？
- 是连续漂移，还是间歇冲击？
- 是设备磨损型，还是气泡/漏箔型？
- 是否已经需要客户汇报或开 FACA？
- 验证计划该看哪些指标？

这个小工具把这些问题变成一个可视化实验，适合用于：

1. 工程师培训；
2. 异常复盘；
3. 现场快速数据预判；
4. GitHub 项目展示；
5. 后续扩展成 Electron 桌面程序或 Web App。

## 快速开始

直接打开：

```text
coating-cpk-lab/index.html
```

不需要 Node.js，不需要 Python，不需要安装依赖。

## CSV 格式

支持导入至少包含以下列名之一的 CSV：

```csv
time,value
2026-05-30 08:00:00,63.21
2026-05-30 08:01:00,63.15
```

也支持使用 `idx,value` 或只保留第二列数值。
